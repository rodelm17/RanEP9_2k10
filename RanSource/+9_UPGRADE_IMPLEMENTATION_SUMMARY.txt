**What It Does**:
1. **Detects all equipment pieces** with valid files
2. **Sets grind level to 9** for each piece
3. **Logs the upgrade process** for debugging
4. **Applies to ALL background characters** from log_in.wld

**Code Snippet**:
```cpp
// AUTO +9 UPGRADE FOR BACKGROUND CHARACTERS
// Set +9 upgrade level for all equipment pieces that exist
CDebugSet::ToLogFile( "  - Applying +9 upgrades to equipment pieces..." );

for ( int i = 0; i < PIECE_SIZE; i++ )
{
    if ( m_pSkinChar->GetPiece(i) && m_pSkinChar->GetPiece(i)->GetFileName() && strlen(m_pSkinChar->GetPiece(i)->GetFileName()) > 0 )
    {
        // Get current grind level
        int currentLevel = m_pSkinChar->GetPiece(i)->GetGrindLevel();
        CDebugSet::ToLogFile( "    Piece %d: %s (Grind Level: %d -> 9)", i, m_pSkinChar->GetPiece(i)->GetFileName(), currentLevel );
        
        // Set to +9
        m_pSkinChar->GetPiece(i)->SetGrindLevel(9);
    }
}
```
